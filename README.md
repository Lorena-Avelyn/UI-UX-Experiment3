## Aim:

To design and develop a user-friendly food delivery mobile application interface by integrating the best features of popular apps like Swiggy, Zomato, and Uber Eats using Figma, including a Logo/Splash Screen, Home Screen, and Checkout Screen with improved usability, navigation, live tracking, offers, and seamless ordering experience.

---

## Algorithm:

1. Analyze existing food delivery applications such as Swiggy, Zomato, and Uber Eats to identify their strengths, features, and usability patterns.

2. Perform heuristic evaluation to identify UX issues and collect the best interface elements from each application.

3. Create a new food delivery app concept named “Flavoro” with a clean and modern user interface.

4. Design the Logo/Splash Screen by including:

   * App logo and branding
   * Tagline (“Good Food. Great Mood.”)
   * Key features such as fast delivery, best offers, and live tracking
   * Get Started and Login options

5. Design the Home Screen by integrating:

   * Search bar for restaurants and food items
   * Location and notification features
   * Promotional banners and discount offers
   * Food category shortcuts (Pizza, Burgers, Biryani, Healthy Food, etc.)
   * Popular restaurants section with ratings and delivery time
   * Bottom navigation bar for Home, Search, Orders, Offers, and Profile

6. Design the Checkout Screen with:

   * Order summary and item details
   * Quantity adjustment options
   * Coupon application section
   * Delivery address management
   * Multiple payment methods such as UPI, Card, Wallet, and Cash
   * Final order amount and Place Order button

7. Apply UX design principles such as consistency, accessibility, readability, visual hierarchy, and minimalist design throughout the interface.

8. Use proper spacing, icons, typography, and color combinations in Figma to improve user interaction and usability.

9. Connect all screens using prototype interactions in Figma for smooth navigation between pages.

10. Test the prototype to ensure proper usability, easy navigation, and efficient food ordering experience.
